# C++ Learning Notes

This folder contains comprehensive notes for all topics covered in the CPPLearn workspace.

## Structure

The notes are organized to mirror the code structure:

### Module 1 - Fundamentals

- **01_Arrays**: Array operations, searching, sorting, matrices
- **02_Functions**: Function basics, parameter passing, applications
- **03_Data**: Classes, objects, data structures
- **04_Conditionals**: Control flow, decision making
- **05_Mathematical**: Mathematical computations and series
- **06_Recursion**: Recursive algorithms and problem-solving
- **07_Structure**: Struct definitions and usage
- **08_Pointers**: Pointer fundamentals and dynamic memory
- **09_FileHandling**: File I/O operations
- **10_Preprocessor**: Macros, directives, conditional compilation

### Module 2 - Advanced Concepts

- **11_OOPS**: Object-Oriented Programming fundamentals
- **12_Memory**: Memory management, new/delete operators
- **13_Encapsulation**: Data hiding and access control
- **14_Abstraction**: Abstract classes and interfaces

## How to Use

Each folder contains markdown files explaining the concepts with:

- Theory and definitions
- Key concepts
- Code examples
- Best practices
- Common pitfalls
