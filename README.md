# C++ Learning Projects

A well-organized collection of C++ programs covering fundamental to advanced concepts.

## üìÅ Folder Structure

### **01_Arrays/**

Array manipulation and matrix operations

- `array_basics.cpp` - Basic array operations and manipulation
- `array_searching.cpp` - Linear and binary search implementations
- `array_sorting.cpp` - Sorting algorithms (bubble, selection, insertion)
- `matrix_operations.cpp` - Matrix addition, subtraction, multiplication, and transpose

### **02_Functions/**

Function-based programming and modular design

- `temperature_converter.cpp` - Fahrenheit to Celsius converter
- `temperature_converter_v2.cpp` - Enhanced temperature conversion
- `banking_system.cpp` - Banking operations (deposit, withdraw, check balance)

### **03_OOP/**

Object-Oriented Programming concepts

- `student_class.cpp` - Student management using classes
- `book_management.cpp` - Book library system with OOP principles

### **04_Conditionals/**

Control flow and decision-making programs

- `calculator.cpp` - Four-operation calculator with input validation
- `shipping_cost.cpp` - Regional shipping cost calculator
- `grade_calculator.cpp` - Student grade evaluation system
- `student_performance.cpp` - Performance analysis with grading

### **05_Mathematical/**

Mathematical computations and series

- `exponential_series.cpp` - e^x series expansion calculation
- `sine_series.cpp` - sin(x) Taylor series implementation

### **06_Recursion/**

Recursive algorithms and problem-solving

- `directory_traversal.cpp` - Recursive file system navigation
- `staircase_problem.cpp` - Dynamic staircase climbing problem
- `staircase_problem_v2.cpp` - Optimized version with memoization

### **build/**

Compiled executables and binary files

### **misc/**

Test files, old versions, and miscellaneous code
